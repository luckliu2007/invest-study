# 私募基金设立检查清单（中国，教育用途）

> ⚠️ 教育/学习用途，非法律意见。实际操作请咨询持牌律师与基金业协会（AMAC）最新规则。

## 一、主体设立
- [ ] 确定组织形式：公司型 / 合伙型（通常 GP-LP 有限合伙）
- [ ] 核名 & 工商注册（市场监管局）
- [ ] 开立银行基本户 & 募集结算资金账户（监管行）
- [ ] 确定注册地（考虑税收返还政策）

## 二、协会备案
- [ ] 申请私募基金管理人登记（AMAC 资产管理业务综合报送平台）
- [ ] 实缴资本金证明、高管资质（基金从业资格）
- [ ] 内控制度文件（风险管理、合规、信息披露）
- [ ] 法律意见书（律师事务所出具）

## 三、产品设计 & 募集
- [ ] 基金合同 / 合伙协议
- [ ] 私募备忘录（PPM，见 PPM_template.md）
- [ ] 风险揭示书 & 投资者适当性匹配
- [ ] 募集完毕 20 个工作日内备案产品

## 四、运作 & 信披
- [ ] 季度/年度报告（AMAC 信披系统）
- [ ] 投资者信息披露（月报/季报）
- [ ] 审计 & 税务申报
- [ ] 重大事项变更报送
